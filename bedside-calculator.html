How to use

Enter values for any subset of inputs. Results update live and persist between visits. Each card shows:

  • Result + unit (large), with a colored severity bin where applicable.
  • Symbolic formula (italic; rendered with KaTeX where possible).
  • Worked substitution (monospace) once the inputs are filled.
  • One-line interpretation in clinical context — green = within target, amber = borderline, red = breach.

The LPV bundle row at the top gives an at-a-glance pass/fail on VT/kg, Pplat, ΔP, and MP — the four lung-protective targets.

Use the Snapshots panel to save the current input set and compare against later changes (PEEP titration, SBT, prone) — each row shows deltas from the most recent live values.

Optional inputs (ETCO₂, target PaCO₂, cardiac output, MAP, esophageal pressures) unlock additional calculations: Vd/Vt, recommended VE, DO₂, OI, and transpulmonary pressures.

Reference values. PBW (men) = 50 + 2.3·(ht in − 60); women = 45.5 + 2.3·(ht in − 60). A–a normal = age/4 + 4 (room air; underestimates on supplemental O₂). MP (VC) = 0.098 × RR × VT × (PIP − ½ ΔP); MP (PC) = 0.098 × RR × VT × (PEEP + ΔP). CaO₂ = 1.34·Hb·SaO₂ + 0.003·PaO₂. ROX = (SpO₂/FiO₂)/RR. Vd/Vt = (PaCO₂ − ETCO₂)/PaCO₂. OI = MAP × FiO₂ × 100 / PaO₂. DO₂ = CO × CaO₂ × 10. PL,insp = Pplat − Pesoph,insp; PL,exp = PEEP − Pesoph,exp. Raw = (PIP − Pplat) / flow_L/s (constant-flow VC). τ = Raw × Cstat (in L/cmH₂O); expiratory Te ≥ 3τ avoids air-trapping.
